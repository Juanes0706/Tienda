Planificación del Proyecto

Objetivos del Proyecto

  1. Desarrollo de una API RESTful

    • Implementar endpoints para CRUD de categorías, productos, clientes y ventas.
    • Utilizar FastAPI para el backend.
    • Integrar base de datos asíncrona con SQLAlchemy.

  2. Interfaz de Usuario

    • Crear páginas HTML con formularios para operaciones CRUD.
    • Implementar navegación con navbar y dropdowns.
    • Diseñar con CSS responsivo.

  3. Gestión de Datos

    • Implementar soft delete para mantener historial.
    • Validar datos de entrada.
    • Manejar errores y excepciones.

Tecnologías Utilizadas

  • Backend: FastAPI, SQLAlchemy, AsyncPG
  • Frontend: HTML, CSS, JavaScript
  • Base de Datos: PostgreSQL
  • Despliegue: Uvicorn, Docker
