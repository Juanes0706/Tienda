Información del Proyecto

Descripción

Este proyecto es una aplicación web para la gestión de inventario de una tienda. Permite administrar categorías, productos, clientes y ventas de manera eficiente.

Funcionalidades

  • Gestión completa de categorías (CRUD)
  • Administración de productos con control de stock
  • Registro y seguimiento de clientes
  • Procesamiento de ventas con detalles
  • Historial de eliminaciones (soft delete)
  • Interfaz web responsiva

Tecnologías Utilizadas

  • Backend: FastAPI, SQLAlchemy, AsyncPG
  • Frontend: HTML5, CSS3, JavaScript
  • Base de Datos: PostgreSQL
  • Despliegue: Uvicorn, Docker
  • Control de Versiones: Git

Requisitos del Sistema

  • Python 3.8+
  • PostgreSQL
  • Navegador web moderno

Instalación

  1. Clonar el repositorio
  2. Crear un entorno virtual
  3. Instalar dependencias: pip install -r requirements.txt
  4. Configurar la base de datos
  5. Ejecutar la aplicación: uvicorn main:app --reload
